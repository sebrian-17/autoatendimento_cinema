# Sistema de Autoatendimento: CineMaster 3000

Sistema de autoatendimento para cinema desenvolvido em Python, simulando a compra e validação de ingressos.

## Funcionalidades

- Compra de ingressos
- Validação de classificação indicativa
- Adição de pipoca ao pedido
- Aplicação de cupom promocional
- Pagamento via cartão de crédito
- Validação de ingressos adquiridos externamente

## Conceitos utilizados

- Estruturas condicionais (if / elif / else)
- Laços de repetição (while)
- Tratamento de exceções (try / except)
- Manipulação e normalização de strings (strip, lower, upper, title)
- Validação de entradas do usuário
- Controle de fluxo do programa
- Simulação de regras de negócio
- Uso de variáveis de controle de estado
- Formatação de saída no terminal
